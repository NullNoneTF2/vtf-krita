# Code of Conduct

This project is meant to be a simple space to simply work on this plugin.

## Our expectations

- Be respectful and kind in discussion.
- Assume good intent and communicate clearly.
- Avoid harassment, insults, or discriminatory language.
- Keep feedback constructive and focused on the work.

## What happens if someone violates this

If a report is made, maintainers may take any action they deem appropriate, including warning the person, limiting interaction, or removing problematic content.

## Scope

This code of conduct applies to all project spaces, including issues, pull requests, discussions, and repository activity.

## Contact

If you have concerns, please contact the project maintainer directly.
